# 🚢 Titanic Dataset - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Titanic dataset to uncover patterns and factors that influenced passenger survival. The analysis includes data cleaning, visualization, and statistical insights using Python.

## 📊 Features

- Load and explore the Titanic dataset
- Handle missing values
- Analyze passenger survival distribution
- Compare survival by gender and passenger class
- Visualize age and fare distributions
- Generate correlation heatmap for numerical features

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Dataset

**Dataset Used:**
- `train.csv` (Titanic Dataset)

## 📈 Visualizations

- Survival Count
- Survival by Gender
- Survival by Passenger Class
- Age Distribution
- Fare Distribution
- Correlation Heatmap
